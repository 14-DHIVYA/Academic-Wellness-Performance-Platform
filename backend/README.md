# 🎓 Academic Wellness Performance Platform – Backend

This is the backend API for the **Academic Wellness Performance Platform**, designed to help track students’ academic habits and wellness metrics securely and efficiently.

---

## 🚀 Features

- User Authentication using JWT
- Protected routes (users can access only their own data)
- Academic & Wellness data management
- Pagination support for large datasets
- Centralized error handling
- Health check API for server monitoring

---

## 🛠️ Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB + Mongoose**
- **JWT Authentication**
- **Postman** (API testing)
